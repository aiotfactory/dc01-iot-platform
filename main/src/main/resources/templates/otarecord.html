OTA Details

ID:

The unique identifier for this OTA firmware record.

Name:

The name of the firmware update, typically representing the purpose.

Comments:

The name of the firmware update, typically representing the purpose.

OTA Target Device Type:

The type of device targeted by this firmware update, e.g., "DC01".

OTA Target Versions:

The specific version(s) of the firmware that the devices will be upgraded to.

OTA Version:

The current version of the firmware to be deployed.

OTA Create Time:

The timestamp indicating when the OTA record was created.

OTA Modify Time:

The timestamp indicating when the OTA record was last modified.

OTA Upgrade Scope:

Defines the scope of devices to which the firmware will be applied, e.g., "ALL (Update regardless of whether the device No is in the include or exclude list.)", "NONE (Won't upgrade)", "INCLUDE ONLY (Only upgrade device in the 'include' list)", "EXCEPT EXCLUDE (Update all devices except those in the exclude list)".

OTA Upgrade Strategy:

The strategy for upgrading devices, such as "IGNORE SAME (it won't upgrade if version of firmware in the device is same as the OTA version')", "FORCE SAME (it will upgrade if version of firmware in the device is same as the OTA version')".

OTA Download Times:

The number of times the firmware has been downloaded by devices.

Firmware Name:

The name of the firmware file.

Firmware Original Name:

The original name of the firmware when user uploads.

Firmware File ID:

A unique identifier for the firmware file in the system.

Firmware Size:

The size of the firmware file in bytes.

Firmware MD5:

The MD5 hash of the firmware file, used for integrity verification.

Firmware Upload Time:

The timestamp when the firmware file was uploaded to the server.

Included Devices:

The list of device IDs that are included in the firmware update.

Excluded Devices:

The list of device IDs that are excluded from this firmware update.
