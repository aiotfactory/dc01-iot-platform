用户体系

用户注册

用户可以自行注册也可以由父用户创建，建议用户名为email，否则无法找回密码。父用户可以创建子用户，但子用户无法再创建子用户。

用户种类

用户分为父用户和子用户，父用户可以管理子用户，也可以查看子用户下面的设备。

用户克隆

比如父用户创建了a1,a2,b两个子用户，但希望b用户可以查看但不能修改a1和a2用户的设备，那么可以设置b用户"克隆自"a1,a2用户，多个用户通过逗号分割。 只有父用户能操作自己的子用户。

用户状态

ENABLED, DISABLED, FREEZED, WAIT_ACTIVATION 分别表示正常，禁用，冻结和等待激活，父用户可以设置子用户的状态。

用户级别

PERSONAL, PROFESSIONAL, ENTERPRISE, ADMIN 分别表示个人用户，专业用户，企业用户和管理员，其中管理员可以执行所有管理操作，可以改变其他用户级别， 不同级别对应权限和功能不同，具体参考。
