MODULE_FORWARD module introduction

Module Description

To facilitate customer expansion functions, this module is provided. Users can execute commands through the WEB page or API call methods. The command text will be sent to the device. Currently, there is no processing on the device. Customers can modify the C language code (forward_util.c) on the device to process it and then return custom content. After the returned content is reported to the cloud, the cloud does not perform any business processing but only forwards it to the caller.
