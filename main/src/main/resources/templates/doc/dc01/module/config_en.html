MODULE_CONFIG module introduction

Module Description

After the device is started and the network connection is established, it will first report the configuration information on the device once. If users also want to have the device periodically report configuration information, they can enable this module. Even if this module is disabled, the device will still report the configuration information once after establishing the connection for the first time. If the network is interrupted in the middle, when the connection is re-established, it will report once again. At the beginning of each module page, there are configurations related to that module, while the MODULE_CONFIG module allows you to view all configurations, making it convenient for users to handle them centrally. Click here to view all configuration descriptions. The MODULE_CONFIG section introduces how to manage this module.
