MODULE_4G module introduction

Module Description

The DC01 product has a 4G LTE module. The international version supports global standards, while the Chinese mainland version only supports cards from China Mobile. Click here to view and configure the MODULE_4G section.

Runtime Information

Each time the device restarts, it will obtain a Csq value which is used to evaluate the strength of the 4G network signal. Generally, a value above 20 indicates good reception, while below 10 may indicate poor reception.

Csq Min and Csq Max are stored as minimum and maximum values in the cloud.

Other status values are periodically reported by the MODULE_META module, so this module must be enabled to access these data.

Iccid: The ICCID number of the SIM card on the device; Imei: The IMEI number of the 4G module, usually printed on the surface of the 4G module; Ip, Mask, Gateway, Dns1, Dns2, Ipv6: Network configuration obtained by the 4G module from the operator's base station.
