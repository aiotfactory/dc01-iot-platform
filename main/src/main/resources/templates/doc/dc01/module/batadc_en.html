MODULE_BATADC Module Introduction

Module Description

The device can automatically measure the input battery voltage. The high and low voltage levels can be used to evaluate the remaining battery power. It supports automatic data reporting. Through command requests for reporting and clicking 'Refresh' next to 'Runtime Information', three forms of reported data have the same format.

Automatic Data Reporting

The meaning of the fields is relatively clear. You can also click here to view the MODULE_BATADC module, which introduces the meanings of the reported data fields.

Command-Based Data Reporting

Sending commands to the device will also trigger the device to report data. You can click here to view the related chapters in the MODULE_BATADC module to understand what commands can be executed and the format of the reported data.
