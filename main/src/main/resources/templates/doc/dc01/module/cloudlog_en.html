MODULE_CLOUDLOG Module Introduction

Module Description

When the device is running, it will output logs. Generally, in embedded development, you need to connect your computer and the device with a specified line to see the logs on the computer. This system provides cloud-based logging, where the device uploads logs to the cloud via the network, allowing you to view the logs directly from the cloud without needing to connect to the device. However, this increases the usage of traffic. Click here to view the configuration for enabling or disabling cloud-based logging.

Runtime Information

Log Status example {"times":135,"uploadTime":"2025-02-18T10:36:38.036+00:00"}, indicating the cumulative number of uploaded logs and the most recent upload time.
