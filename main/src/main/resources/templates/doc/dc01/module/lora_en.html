MODULE_LORA Module Introduction

Module Description

The DC01 product has an LORA module that can be set to either transmit or receive data. For more details, please refer to the LORA section.

Automatic Data Reporting

Automatically reported data includes the status information of the LORA operation. This mainly refers to the number of times Tx and Rx have occurred as well as the number of errors encountered. Interrupt reporting actually means that when the LORA is in listening mode and receives information, it automatically reports it. For more details, please refer to the LORA section.

Interrupt Data Reporting

Interrupt reporting actually means that when the LORA is in listening mode and receives information, it automatically reports it. For more details, please refer to the LORA section.

Command Data Reporting

Data can be triggered for transmission by sending commands to the device. Please refer to the LORA section to understand which commands can be executed and the format of the reported data.
