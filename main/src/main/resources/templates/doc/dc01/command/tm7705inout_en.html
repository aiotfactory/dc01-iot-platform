Description

After configuring the TM7705, the device will periodically report data, and you can also read data at any time through commands; these two methods do not conflict.

Parameter Description
Parameter Description Necessary Type Bytes In/Out
operate Fixed to 14 Yes Number 1 Input
pin 1 indicates reading the value of pin 1, 2 indicates reading the value of pin 2, 3 indicates reading both values simultaneously. Yes Number 1 Input
Example
Read the value of pin 1 amplified by 10^7 times, so divide by 10^7 to get the voltage value in units of volts, if the voltage value is -100, it indicates an error in reading:

operate=14 pin=1
