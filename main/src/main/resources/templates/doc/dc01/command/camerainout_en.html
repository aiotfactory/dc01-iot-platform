Description

The device can be configured to periodically upload captured images, or you can use this command to temporarily retrieve an image.

Parameter Description
Parameter Description Required Type Bytes Input/Output
operate Fixed value: 15 Yes Number 1 Input
Examples
Capture an image and return the image content in base64 format:

operate=15

Response

picName The generated image name, also stored in the cloud.
picContentBase64 Since JSON cannot transmit binary data, the image content is encoded in base64 format.
picId The ID of the image stored in the cloud.
info Indicates successful retrieval.
