说明

模块用于检测温湿度，因为模块是预装在线路板上，线路板运行时温度会有所升高，因此测到的温度比环境要高一些。

参数说明
参数 描述 必须 类型 字节 入出
operate 固定为19 是 数字 1 输入
例子
获取温湿度:

operate=19

返回，正常

temperature 本次测到的温度，除以100后为摄氏度。
humidity 本次测到的湿度，除以100后为湿度的百分比值。
temperatureMax,temperatureMin 设备开机以来，测到的最大和最小温度。
humidityMax,humidityMin 设备开机以来，测到的最大和最小湿度。
timesTotal,timesSuccess 设备开机以来，测量的次数和成功次数。
flag 为0表示本测量正常，其他值表示测量不正常。
status 从AHT20读取到的状态字节，每次读取7字节数据，这个是第0个字节。

返回正常
