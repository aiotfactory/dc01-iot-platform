说明

线路板上有LORA，TM7705等SPI模块，而且线路板也可以外接其他SPI模块，通过本命令可以了解SPI相关设备运行的状况。

参数说明
参数 描述 必须 类型 字节 入出
operate 固定为4 是 数字 1 输入
例子
获取SPI运行状态:

operate=4

返回

spi_bus 1表示初始化成功，0表示不成功
0.noname:inited 0表示SPI设备0，目前共支持5组SPI设备，编号为0-4，0用于板子上LORA模块，1用于板子上TM7705传感器，2用于板子上W5500芯片，3可以外接其他模块，4也可以外接其他模块。
0.noname:inited name表示初始化每个模块时是否取名，没有取名默认为noname。
0.noname:inited=0 表示SPI设备编号0，也就是LORA模块，没有初始化或者说没有开启。
0.noname:inited=0,tx=0,rx=0,fail=0; tx，rx表示发送和接收的次数，fail表示失败的次数，因为没有初始化，自然都是0。
1.not set:inited=1,tx=12,rx=17,fail=0; SPI编号1，也就是TM7705，已经初始化，并且发送12次，接收17次，失败0次。
