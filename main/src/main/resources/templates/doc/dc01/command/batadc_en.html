Description

When the device is powered by a battery, it is necessary to monitor the battery level. If the battery is low, a reminder to charge or replace the battery should be triggered. A simple approach is to measure the battery voltage. For example, for a regular polymer lithium battery, charging is required when the voltage drops below 3.5V. Through experiments, a relationship table between voltage and remaining battery capacity can also be established.

Parameter Description
Parameter Description Required Type Bytes Input/Output
operate Fixed value: 12 Yes Number 1 Input
Examples
Get Battery Voltage:

operate=12

Response, Normal

adcTimes Indicates how many times the battery voltage has been measured since startup.
adcTimesFailed Indicates the number of failed measurement attempts since startup.
adcPinOnoff To measure the voltage, the battery voltage is divided and connected to an ESP pin. This results in a small current loss of a few microamperes. In scenarios where extreme power saving is critical, we use a switch to control this connection. If the switch is off, the microampere-level current loss will not occur.
adcValue This is the result of multiplying millivolts (mV) by 10000. Therefore, dividing this value by 10000 gives the voltage in mV.
