Description

Read the values of all GPIOs at once.

Parameter Description
Parameter Description Required Type Bytes Input/Output
operate Fixed value: 7 Yes Number 1 Input
Examples
Read the operating status. Only pins configured as enabled are returned; pins configured as 0 are not returned:

operate=7
