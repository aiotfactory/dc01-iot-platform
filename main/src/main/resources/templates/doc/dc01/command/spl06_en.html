Description

The module is used to detect air pressure but can also detect temperature.

Parameter Description
Parameter Description Necessary Type Bytes In/Out
operate Fixed at 20 Yes Number 1 Input
Example
Get Air Pressure and Temperature:

operate=20

Return, Normal

temperature The measured temperature this time, divided by 100 for Celsius.
pressure The measured air pressure this time, divided by 100 for Pa.
temperatureMax,temperatureMin The maximum and minimum temperatures since the device was turned on.
pressureMax,pressureMin The maximum and minimum air pressures since the device was turned on.
timesTotal,timesSuccess The total number of measurements and successful measurements since the device was turned on.
flag Value 0 indicates that the measurement is normal; other values indicate abnormal measurement.

Return Normal
