Description

Retrieve device operational data. This command fetches various operational data of the device, such as IMEI, ICCID, etc.

Parameter Description
Parameter Description Required Type Bytes Input/Output
operate Fixed value: 13 Yes Number 1 Input
Examples
Retrieve Device Operational Status:

operate=13

Response

imei The IMEI serial number of the 4G module.
iccid The ICCID of the SIM card in the 4G module.
aliveRunSeconds The number of seconds the device has been running since startup.
aliveTimesReconn The number of times the network was reconnected since the device started.
aliveTimesMeta The number of times status data was reported since the device started.
runtimeRestartReason If a controllable exception occurs (note that uncontrollable exceptions like power loss or errors may occur which cannot be captured or logged in time), the device will record the reason in flash memory, which can be read after the next startup but is not cleared. This means the value could have been written during the previous or earlier runs. Values: 0 = unknown, 1 = 4G modem error 1, 2 = 4G modem error 2, 3 = config change, 4 = user_malloc error, 5-8 = SPI error.
runtimeRestartReasonSeconds The runtime duration of the device when runtimeRestartReason was recorded.
runtimeRestartReasonTimes The number of reboots when runtimeRestartReason was recorded. The device maintains a counter in flash memory, incrementing it automatically each time it starts, indicating how many times it has restarted.
runtimeRestartTimes The device maintains a counter in flash memory, incrementing it automatically each time it starts, indicating how many times it has restarted.
runtimeResetReason Each time the device restarts, ESP determines a reason: 0 = Reset reason cannot be determined, 1 = Reset due to power-on event, 2 = Reset by external pin (not applicable for ESP32), 3 = Software reset via esp_restart, 4 = Software reset due to exception/panic, 5 = Reset (software or hardware) due to interrupt watchdog, 6 = Reset due to task watchdog, 7 = Reset due to other watchdogs, 8 = Reset after exiting deep sleep mode, 9 = Brownout reset (software or hardware), 10 = Reset over SDIO, 11 = Reset by USB peripheral, 12 = Reset by JTAG, 13 = Reset due to efuse error, 14 = Reset due to power glitch detected, 15 = Reset due to CPU lockup.
module4gIp The IP address obtained by the 4G module.
module4gMask The subnet mask obtained by the 4G module.
module4gGw The gateway obtained by the 4G module.
module4gDns1 The primary DNS obtained by the 4G module.
module4gDns2 The secondary DNS obtained by the 4G module.
module4gIpv6 The IPv6 address obtained by the 4G module. If the carrier does not support IPv6, this value will not be available.
moduleWifiStaIp If the device connects to another Wi-Fi router as a station (STA), this is the IP address obtained.
moduleWifiStaMask If the device connects to another Wi-Fi router as a station (STA), this is the subnet mask obtained.
moduleWifiStaGw If the device connects to another Wi-Fi router as a station (STA), this is the gateway obtained.
moduleWifiApDns If the device connects to another Wi-Fi router as a station (STA), this is the DNS used.
rt_wifi_ap_sta_num If the device operates as an access point (AP), similar to a router mode, this value represents the number of Wi-Fi terminals (STAs) connected to the device.
moduleW5500Ip If the device connects to another gateway or router via Ethernet cable, this is the IP address obtained.
moduleW5500Mask If the device connects to another gateway or router via Ethernet cable, this is the subnet mask obtained.
moduleW5500Gw If the device connects to another gateway or router via Ethernet cable, this is the gateway obtained.
moduleW5500Dns1 If the device connects to another gateway or router via Ethernet cable, this is the primary DNS obtained.
moduleW5500Dns2 If the device connects to another gateway or router via Ethernet cable, this is the secondary DNS obtained.
moduleW5500Ipv6 If the device connects to another gateway or router via Ethernet cable, this is the IPv6 address obtained. If IPv6 is not supported, this value will not be available.
