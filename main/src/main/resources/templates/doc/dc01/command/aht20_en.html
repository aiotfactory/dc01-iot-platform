Description

The module is used to detect temperature and humidity. Since the module is pre-installed on the circuit board, the temperature of the circuit board will increase slightly during operation, so the measured temperature will be slightly higher than the ambient temperature.

Parameter Description
Parameter Description Required Type Bytes Input/Output
operate Fixed value: 19 Yes Numeric 1 Input
Example
Get Temperature and Humidity:

operate=19

Response, Normal

temperature The measured temperature; divide by 100 to get the value in Celsius.
humidity The measured humidity; divide by 100 to get the percentage value.
temperatureMax, temperatureMin The maximum and minimum temperatures measured since the device was powered on.
humidityMax, humidityMin The maximum and minimum humidity values measured since the device was powered on.
timesTotal, timesSuccess The total number of measurements and successful measurements since the device was powered on.
flag A value of 0 indicates normal measurement; any other value indicates abnormal measurement.
status The status byte read from the AHT20 sensor; this is the 0th byte when reading 7 bytes of data.
