说明

配置好TM7705后，设备会周期上报数据，也可以通过命令随时读取数据，两者不冲突。

参数说明
参数 描述 必须 类型 字节 入出
operate 固定为14 是 数字 1 输入
pin 1表示读取管脚1的值，2表示读取管脚2的值，3表示同时读取2个的值。 是 数字 1 输入
例子
读取管脚1，pin1的值放大了10的7次方倍，因此除以10的7次后的值为单位福特的电压值，如果电压值为-100，表示读取出错:

operate=14 pin=1

返回

{"id":"67ac7dda8545421273252f37","deviceId":"67ac4240c2fb41360998f33d","moduleTypeId":5,"request":{"rawString":"operate=14 pin=1","sourceType":"CLOUD_COMMAND","ip":"127.0.0.1"},"requestTime":"2025-02-12T10:54:17.230674654Z","upload":{"pin1Value":2145,"pin2Value":null},"uploadTime":"2025-02-12T10:54:18.076961433Z","command":3,"operate":14,"info":null,"errorType":"OK","dataCommType":"REQUEST_UPLOAD","dataCommSource":"CLOUD_COMMAND"}

读取管脚1，2，pin1，pin2的值放大了10的7次方倍，因此除以10的7次后的值为单位福特的电压值，如果电压值为-100，表示读取出错:

operate=14 pin=3

返回
