说明

当设备通过电池供电时，需要检测电池电量，如果低电量要提醒充电或者更换电池，比较简单的做法是检测电池的电压，比如普通聚合物锂电池，电压低于3.5V就要充电了，通过实验，也可以制定一个电压与剩余电量的关系表。

参数说明
参数 描述 必须 类型 字节 入出
operate 固定为12 是 数字 1 输入
例子
获取电池电压:

operate=12

返回，正常

adcTimes 表示开机后测量了多次电池电压。
adcTimesFailed 表示开机后测量失败次数。
adcPinOnoff 测量电压需要把电池分压后，接到ESP的引脚，这会导致几个微安的电流损失，在极度苛求省电的场景，我们通过一个开关来控制这个连接，如果开关断开，就不会消耗这几个微安的电流。
adcValue 这是mV乘以10000的结果，因此该值除以10000所得值的单位为mV。
