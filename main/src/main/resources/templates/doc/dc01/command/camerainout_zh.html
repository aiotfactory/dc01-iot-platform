说明

设备可以配置为周期上传拍照图片，也可以通过本命令临时读取图片。

参数说明
参数 描述 必须 类型 字节 入出
operate 固定为15 是 数字 1 输入
例子
拍照，并以base64格式返回图片内容:

operate=15

返回

picName 生成的图片名称，同时存在云端。
picContentBase64 因为Json不能传输二进制数据，因此图片内容以base64方式编码。
picId 存在云端的图片ID。
info 表示读取成功。
