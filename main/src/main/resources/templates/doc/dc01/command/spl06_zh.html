说明

模块用于检测气压，但同时也能检测温度。

参数说明
参数 描述 必须 类型 字节 入出
operate 固定为20 是 数字 1 输入
例子
获取气压和温度:

operate=20

返回，正常

temperature 本次测到的温度，除以100后为摄氏度。
pressure 本次测到的气压，除以100后为Pa。
temperatureMax,temperatureMin 设备开机以来，测到的最大和最小温度。
pressureMax,pressureMin 设备开机以来，测到的最大和最小气压。
timesTotal,timesSuccess 设备开机以来，测量的次数和成功次数。
flag 为0表示本测量正常，其他值表示测量不正常。
